ACT Theatre rebranded as Union Arts Center, now at `https://www.unionartscenter.org`.
Ticketing via Tessitura at `https://order.unionartscenter.org/events`. The ticketing
site is an Angular SPA (class prefix `tn-`) that renders events client-side — no
static HTML event data. Tessitura REST API does not expose public unauthenticated
endpoints for event listing at this installation. WordPress main site has no events
post type. Not currently viable without browser rendering.
