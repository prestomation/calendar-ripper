Dance venue in Capitol Hill (Capitol Hill Block Party area). Uses WordPress with
the Modern Events Calendar (MEC) plugin (`mec-theme-eventsia` body class,
`mec_skin_2986` shortcode). The MEC REST API (`/wp-json/mec/v1/events`) returns
an empty array. AJAX endpoint (`wp-admin/admin-ajax.php?action=mec_load_events_sc`)
returns `0` (authentication required). Tribe Events is not installed. No ICS
export found. Would require browser rendering or authentication to access event
data. Not currently viable.

Note: The venue is now called "Century Ballroom Roadshow" (renamed after
reopening at Reverie Ballroom location).
